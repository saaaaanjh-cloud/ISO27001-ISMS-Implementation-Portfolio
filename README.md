# ISO27001-ISMS-Implementation-Portfolio
This repository demonstrates a practical ISO/IEC 27001:2022 aligned Information Security Management System (ISMS) implementation for a simulated Digital Payments & AML Technology organization.
## ISMS Scope
The ISMS includes:
- AML transaction monitoring systems
- Payment processing platforms
- Customer and transaction databases
- Cloud infrastructure (AWS)
- Corporate IT systems
- Third-party vendors supporting operations
## Scope
The ISMS covers:
- AML transaction monitoring systems
- Payment processing platforms
- Cloud-hosted databases
- Employee endpoints
- Third-party service providers
- ## Deliverables
- ISMS Scope & Context
- Risk Assessment Methodology
- Risk Register & Treatment Plan
- Statement of Applicability (SoA)
- Security Policies
- Internal Audit Checklist
- ## Standards Used
- ISO/IEC 27001:2022
- ISO/IEC 27002:2022
- ISO 31000 Risk Management
- ## Disclaimer
This is a self-directed educational project using simulated data. No real organization data is included.
